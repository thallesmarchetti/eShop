# eShop
> Projeto do bloco de Android.

Protótipo de aplicativo Android voltado para uma loja virtual de produtos.

## Funcionalidades

- [x] Sistema de login com Firebase
- [x] Esqueceu a senha
- [x] Cadastro de nova conta
- [x] Logout da conta
- [x] Adicionar Produtos
- [x] Detalhes dos produtos
- [x] Vender produtos
- [x] Comprar produtos
- [x] Editar perfil
